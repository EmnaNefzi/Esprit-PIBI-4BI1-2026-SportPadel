## Overview

This folder contains the Angular frontend application of the **Sport Padel BI** project.

The Angular app is used to visualize and interact with analytical data coming from the Data Warehouse (DW) through the Flask API.  
It provides a modern web interface for dashboards, player analytics, tournament insights, and performance tracking.

---

## Role in the Global Architecture

The Angular application is the presentation layer of the BI architecture.
