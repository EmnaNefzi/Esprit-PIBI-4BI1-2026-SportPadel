## Overview

This folder contains all Power BI files used for data visualization and reporting in the **Sport Padel BI** project.

Power BI is connected directly to the **Data Warehouse (DW)** built in PostgreSQL and populated through the Talend ETL pipeline.

It provides interactive dashboards for:
- Player performance analysis
- Tournament analytics
- Ranking evolution
- Audience and popularity insights
