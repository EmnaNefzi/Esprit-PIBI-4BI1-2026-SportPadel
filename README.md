# Sport Padel – Data Warehouse & Business Intelligence Project

## Overview
This project was developed as part of the PIBI – 4th Year Business Intelligence Program at **Esprit School of Engineering** (Academic Year 2025–2026).

It implements a full Business Intelligence solution for Sport Padel using **Talend ETL**, **PostgreSQL**, and **Power BI dashboards**, with an optional match reservation module using **Angular** frontend and **Flask/Django** REST API.  
The project follows the **GIMSI methodology**, ensuring alignment with real business needs and decision-support requirements.

## Features
- ETL Integration using Talend Open Studio
- Data Cleaning and Transformation
- Data Warehouse Implementation
- Dimensional Modeling (Fact & Dimension Tables)
- Interactive Dashboards with Power BI Desktop
- KPI Monitoring & Drill-downs
- Analytical Reporting
- Optional: Match Reservation Module (Angular + REST API)

## Tech Stack

### Frontend
- Angular (for match reservation module)
- Power BI Desktop (dashboards, KPI monitoring & drill-downs)

### Backend
- Flask / Django (REST API for match reservation module)
- PostgreSQL (Database & Data Warehouse)
- Talend Open Studio (ETL pipeline)
- Apache Airflow (orchestration)

## Architecture
Data Sources → Talend ETL → PostgreSQL Data Warehouse → Power BI Dashboards & Reports  
Optional: Angular frontend + Flask/Django API for Match Reservations

## Getting Started
1. Open Talend Open Studio  
2. Import ETL Jobs  
3. Configure PostgreSQL connection  
4. Execute ETL workflows to populate the Data Warehouse  
5. Connect Power BI to PostgreSQL  
6. Build dashboards and analytical reports  
7. Optional: Run Angular + Flask/Django application for match reservations
