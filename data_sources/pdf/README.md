# PDF Files - Sport Padel BI

This folder contains all the original PDF files used as sources for the Sport Padel BI project.  
These PDFs are used to extract tournament, player, and calendar information which is then converted into CSV files.

## Files Overview
### Padel-Rackets

### Players

### calendar
- `2026-PadelTour.pdf`
- `calendar-2023.pdf`
- `calendar-2024.pdf`


## Usage
These PDFs are processed by Python scripts located in the `../scripts/` folder to generate CSV files in `../csv/`.
